## Tool Clip Hanger
### Description
Tool hanger that utilizes the belt clip that comes with pretty much all cordless tools. We have tested many clips from not only different brands but also aftermarket/generic clips as well.

To mount the tool holder to the wall we have went with a simple deck or drywall screw approach. Any screw that has a shaft diameter of ≤ 4.5mm and a head diameter ≤ 9mm will suffice.
